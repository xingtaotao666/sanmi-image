## 方案一：Vercel 部署（推荐 · 免费 · 最简单）

### 优点
- 免费额度足够个人/小团队使用
- 自动 HTTPS、全球 CDN
- 推送代码自动部署

### 步骤

#### 1. 注册账号
- 打开 https://vercel.com
- 用 GitHub 账号注册（没有就先注册 GitHub）

#### 2. 上传代码到 GitHub
```bash
# 在项目目录初始化 Git
cd 三米生图
git init
git add .
git commit -m "三米生图 v1.0"

# 在 GitHub 创建新仓库 sanmi-image-gen
# 然后推送
git remote add origin https://github.com/你的用户名/sanmi-image-gen.git
git branch -M main
git push -u origin main
```

#### 3. 在 Vercel 导入项目
1. 打开 https://vercel.com/new
2. 选择你的 GitHub 仓库 `sanmi-image-gen`
3. **Framework Preset** 选 `Other`
4. **Build Command** 留空
5. **Output Directory** 填 `public`
6. 点击 `Environment Variables`，添加：
   - `ARK_API_KEY` = `你的API Key`
   - `ARK_MODEL` = `doubao-seedream-5-0-pro-260628`
7. 点击 `Deploy`

#### 4. 创建 API 路由（Vercel Serverless）
在项目根目录创建 `api/generate.js`（Vercel 会自动识别）：
```javascript
// api/generate.js - Vercel Serverless Function
export default async function handler(req, res) {
  if (req.method !== 'POST') {
    return res.status(405).json({ error: 'Method not allowed' });
  }

  const { prompt, size, image, watermark } = req.body;

  const reqBody = {
    model: process.env.ARK_MODEL || 'doubao-seedream-5-0-pro-260628',
    prompt,
    response_format: 'url',
    size: size || '2048x2048',
    stream: false,
    watermark: watermark !== false,
  };

  if (image) reqBody.image = image;

  const response = await fetch('https://ark.cn-beijing.volces.com/api/v3/images/generations', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'Authorization': `Bearer ${process.env.ARK_API_KEY}`,
    },
    body: JSON.stringify(reqBody),
  });

  const data = await response.json();
  res.status(response.status).json(data);
}
```

> 部署后访问 `https://你的项目名.vercel.app` 即可使用。

## 方案二：云服务器部署（适合商用）

### 推荐：腾讯云轻量应用服务器

#### 1. 购买服务器
- 打开 https://cloud.tencent.com/product/lighthouse
- 选择套餐：最低配 2核2G 即可（约 50元/月）
- 系统选 Ubuntu 22.04

#### 2. 连接服务器
```bash
ssh ubuntu@你的服务器IP
```

#### 3. 安装 Node.js
```bash
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs
node -v  # 确认安装成功
```

#### 4. 上传项目
```bash
# 方法一：用 Git
git clone https://github.com/你的用户名/sanmi-image-gen.git
cd sanmi-image-gen
npm install

# 方法二：用 scp 上传
# 在本地执行：
scp -r 三米生图/ ubuntu@你的服务器IP:~/
```

#### 5. 配置环境变量
```bash
cd sanmi-image-gen
cp .env.example .env
nano .env
# 填入你的 API Key，保存退出（Ctrl+O, Enter, Ctrl+X）
```

#### 6. 安装 PM2（进程守护）
```bash
sudo npm install -g pm2
pm2 start server.js --name "sanmi-image"
pm2 save
pm2 startup  # 设置开机自启
```

#### 7. 配置 Nginx 反向代理
```bash
sudo apt install -y nginx
sudo nano /etc/nginx/sites-available/sanmi-image
```

写入以下内容：
```nginx
server {
    listen 80;
    server_name 你的域名或IP;

    client_max_body_size 50M;  # 图生图需要上传大文件

    location / {
        proxy_pass http://localhost:3000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_cache_bypass $http_upgrade;
    }
}
```

启用配置：
```bash
sudo ln -s /etc/nginx/sites-available/sanmi-image /etc/nginx/sites-enabled/
sudo nginx -t          # 测试配置
sudo systemctl reload nginx
```

#### 8. 配置 HTTPS（可选但推荐）
```bash
sudo apt install -y certbot python3-certbot-nginx
sudo certbot --nginx -d 你的域名
```

> 部署后访问 `http://你的域名` 即可使用。

## 常见问题

### Q: 部署后图片生成报错？
A: 检查服务器环境变量 `ARK_API_KEY` 是否正确设置，检查火山引擎账户余额是否充足。

### Q: 图生图上传图片报错？
A: 检查 Nginx 配置中 `client_max_body_size` 是否设置为 `50M` 以上。

### Q: 如何更新代码？
A: 本地修改 → `git push` → Vercel/Zeabur 自动重新部署；云服务器执行 `git pull && pm2 restart sanmi-image`。

### Q: 如何更换 API Key？
A: 修改 `.env` 文件中的 `ARK_API_KEY`，然后重启服务 `pm2 restart sanmi-image`。Vercel/Zeabur 在后台环境变量中修改。

### Q: 数据存储在哪里？
A: 所有生成的图片存储在用户浏览器本地（IndexedDB），不占服务器空间，换设备需要重新生成。

## 域名备案说明

如果你的服务器在中国大陆（腾讯云、阿里云等），需要完成 **ICP 备案**：
1. 在云服务商控制台找到"备案"入口
2. 准备身份证照片、域名证书
3. 按流程提交，一般 7-20 个工作日
4. 备案通过后才能通过域名访问

如果用 Vercel/Zeabur（海外服务器），不需要备案，但访问速度可能稍慢。

## 安全提醒

1. **永远不要**将 `.env` 文件提交到 GitHub（已在 .gitignore 中排除）
2. **永远不要**在前端代码中写 API Key
3. 定期检查火山引擎账单，设置消费上限
4. 如需多用户使用，建议加上登录认证功能
